# Algoritmos e Estruturas de Dados III (AEDS III)

Repositório contendo atividades, trabalhos, implementações e projetos desenvolvidos durante a disciplina **Algoritmos e Estruturas de Dados III (AEDS III)** da Universidade Federal de Alfenas (UNIFAL-MG).

## 📚 Sobre a disciplina

A disciplina de Algoritmos e Estruturas de Dados III aprofunda o estudo de algoritmos e estruturas de dados aplicados à resolução de problemas computacionais, enfatizando eficiência, organização dos dados e análise de desempenho. O objetivo é desenvolver soluções com foco em desempenho e escalabilidade. :contentReference[oaicite:0]{index=0}

## 📖 Conteúdo abordado

- Análise de Algoritmos
- Complexidade de Tempo e Espaço
- Recursão
- Árvores
- Grafos
- Busca em Grafos
- Caminhos Mínimos
- Árvores Geradoras
- Algoritmos Gulosos
- Divisão e Conquista
- Estruturas de Dados Avançadas
- Projetos e Exercícios Práticos
